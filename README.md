# 🛍️ AIShopzone

AIShopzone is an AI-powered e-commerce platform built with Flask and MySQL. It offers secure user authentication, intelligent product recommendations, shopping cart, checkout, order management, and a complete admin dashboard, localized for India with ₹ pricing and GST support.

## ✨ Features

- 🤖 AI-Powered Product Recommendations
- 🛒 Shopping Cart & Checkout
- ❤️ Wishlist & Order History
- 👤 Secure User Authentication
- 📦 Admin Dashboard & Inventory Management
- 🇮🇳 India-Ready (₹ Currency, GST & UPI-Friendly)

## 💻 Tech Stack

- **Backend:** Python, Flask, SQLAlchemy
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, Jinja2
- **AI:** OpenAI / Claude API

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- MySQL
- pip

### Run Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/AIShopzone.git
```

Navigate to the project:

```bash
cd AIShopzone
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
flask --app app run
```

Open your browser:

```
http://127.0.0.1:5000
```
